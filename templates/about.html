Mini Project - AI Based Resume Analyzer

This is a mini project built using Python, Flask, and basic AI concepts. It compares a resume with a job description and gives:

  • Skill match percentage
  • Text similarity using TF-IDF + Cosine Similarity
  • Missing skills
  • Tips to improve your resume
